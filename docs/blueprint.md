# **App Name**: Write & Paid Funnel

## Core Features:

- Landing Page Sections: Creation of a single-page landing experience, complete with hero section, benefits cards, video placeholder, 'What You'll Get' grid, testimonials, and FAQ accordion.
- Pricing Page with Mock Purchase: Development of a pricing page that simulates a one-time payment flow and a demo modal for completing the purchase.
- Dashboard Mock-up with Query Parameter Toggle: Construct a dashboard with locked/unlocked states controlled via URL parameters. Display of curriculum, resources, and account information sections using Bootstrap tabs/pills.
- Legal Pages: Design of basic static legal pages for terms, privacy, and refunds.
- Neon-Glass Styling: Implementation of the provided neon-glass aesthetic via custom CSS. Applies styles such as dark backgrounds, glass cards, neon buttons, and animated underline hover effects.
- Query String Logic: Enables handling of query string parameters to control UI elements, specifically for the dashboard's locked/unlocked state. Also contains demo modal activation for 'Sign In' and 'Buy Now' buttons.

## Style Guidelines:

- Primary color: Electric Blue (#00e5ff) inspired by the need to write which also expresses clarity, paired with a complementary vivid Magenta (#ff4d9d) that enhances visibility in the dark UI.
- Background color: Dark midnight blue (#0b0f17) providing a clean dark canvas for the neon effects.
- Accent color: Deep purple (#6c236c), adds sophistication and highlights calls to action.
- Body and headline font: 'Inter', sans-serif, offering clean readability for main text and titles.
- Use clean and minimalist icons to convey concepts in the benefits and 'What You'll Get' sections. Neon outlines will enhance the aesthetic.
- Employ Bootstrap's grid system to ensure full responsiveness and maintain a consistent layout across all devices.
- Incorporate subtle animations on the landing page to engage users as they scroll.
